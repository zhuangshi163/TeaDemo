TeaCacheSession, manage session inside cache system instead of

default php session system, extremely useful when we need extreme performance and combined with memcache Create your required cache system, pass the cache object to the TeaCacheSession::installOnCache, example:

  1.  $cache = Tea::cache('memcache');
  2.  TeaCacheSession::insallOnCache($cache, 'myPrefixForSessionID');
  3.  $session = Tea::session("mysite");
After that use session variables normally and transparently ;) Add something to session
  1. $session->someVariable = "something";
Get that variable
  1.  $var = $session=>get("someVariable"); (returns "something")
