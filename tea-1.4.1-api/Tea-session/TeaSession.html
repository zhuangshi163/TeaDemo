TeaSession

Description
Description | Vars (details) | Methods (details)

Session class, manage all session options

Call it from:

  1. $session = Tea::session("mysite");
Add something to session
  1. $session->someVariable = "something";
Get that variable
  1.  $var = $session=>get("someVariable"); (returns "something")
