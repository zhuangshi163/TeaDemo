TeaPager

Description
Description | Vars (details) | Methods (details)

A helper class to help generating pagination widget when displaying a list of items.

You can change every component of the pagination widget through CSS. The CSS class name for the components can be changed if needed.

<p>You can send GET, POST, PUT and DELETE requests with TeaRestClient. Method chaining is also supported by this class.</p> Example usage:

  1.  Tea::loadHelper('TeaPager');
  2.  Tea::loadModel('User');
  3.  
  4.  $page = $this->params['pindex'];
  5.  $total = User::getTotal();
  6.  $pager = new TeaPager(Tea::conf()->APP_URL.'main/gallery', $total, 6, 10);
  7.  
  8.  #change the text of next/prev
  9.  //$pager->prevText = 'prev';
  10.  //$pager->nextText = 'next';
  11.  
  12.  #If you don't need the next/prev links
  13.  //$pager->noNextPrev();
  14.  
  15.  #Do the pagination
  16.  $pager->paginate($page);
  17.  
  18.  $limit = $pager->limit;
  19.  Tea::db()->find('User', array('limit'=>$limit));
  20.  
  21.  echo $pager->defaultCss;
  22.  //echo $pager->components['prev_link'];
  23.  echo $pager->output;
  24.  echo $pager->components['jump_menu'];
  25.  echo $pager->components['page_size'];
  26.  //echo $pager->components['next_link'];
