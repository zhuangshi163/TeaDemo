TeaRestClient

Description
Description | Vars (details) | Methods (details) Constants (details)

A REST client to make requests to 3rd party RESTful web services such as Twitter.

You can send GET, POST, PUT and DELETE requests with TeaRestClient. Method chaining is also supported by this class.

Example usage:

  1.  //Example usage in a Tea Controller
  2.  //import the class and create the REST client object
  3.  $client = $this->load()->helper('TeaRestClient', true);
  4.  $client->connect_to("http://twitter.com/direct_messages.xml")
  5.         ->auth('twituser', 'password', true)
  6.         ->get()
  7.  
  8.  if($client->isSuccess()){
  9.       echo "<pre>Received content-type: {$client->resultContentType()}<br/>";
  10.       print_r( $client->result() );
  11.  }else{
  12.       echo "<pre>Request unsuccessful, error {$client->resultCode()} return";
  13.  }
