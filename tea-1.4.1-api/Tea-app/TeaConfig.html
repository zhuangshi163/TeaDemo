TeaConfig

Description
Description | Vars (details) | Methods (details)

TeaConfig contains all the configuration for the system. This class should not be called directly.

Framework settings are all in capital letters. Settings can be accessed via

  1. Tea::conf()->SOME_SETTINGS

You can defined your own setting properties and access them throughout the application. It is recommended to defined your properties in lower case for future compatibility. eg.

  1. $config['my_setting_here'] = 'example';
