TeaSqlMagic

Description
Description | Descendents | Vars (details) | Methods (details) Constants (details)

TeaSqlMagic is a fast ORM tool which handles relational data.

It generates SQL query on the fly and escaping/quoting record values to prevent sql injection. You can do CRUD operations in an easier way with TeaSqlMagic instead of writing queries manually.

Database models does not need to extend any classes with TeaSqlMagic. Database table relationships are defined in a centralized way in SITE_PATH/protected/config/db.conf.php instead of defining them in each Model class.

Unlike most active records, TeaSqlMagic class does not use any php Magic methods in order to gain higher performance.

It does not implements the active record design pattern, a popular Object-Relational Mapping (ORM) technique. However, it can still be used with some flavours of active record features.

TeaSqlMagic is more like a blend between Active record and DataMapper design pattern.

Creating an instance of an TeaSqlMagic class does not immediately connect to the RDBMS server. Use connect() or reconnect() to make the actual connection on demand.

Please check the Guide for more details about this class.
