TeaDbExpression

Description
Description | Vars (details) | Methods (details)

TeaDbExpression represents a DB expression that does not need escaping.

TeaDbExpression is mainly used with TeaSqlMagic as attribute values. When inserting or updating a database record, attribute values of type TeaDbExpression will be directly put into the corresponding SQL statement without escaping. A typical usage is that an attribute is set with 'NOW()' expression so that saving the record would fill the corresponding column with the current DB server timestamp.

  1.  Tea::loadCore('db/TeaDbExpression');
  2.  $usr = new User;
  3.  $usr->create_date = new TeaDbExpression('NOW()');
  4.  //$usr->create_date = 'NOW()';  will insert the date as a string 'NOW()' as it is escaped.
  5.  Tea::db()->insert($usr);
