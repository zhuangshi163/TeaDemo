TeaFrontCache

Description
Description | Methods (details)

TeaFrontCache provides frontend caching utilities.

It can be retrieved with the shorhand Tea::cache('front'). The frontend cache supports full page and partial page caching mechanism. Cache files are store in the path defined in $config['CACHE_PATH'].

You can start the caching before displaying the view output.

  1.  //Display cache if exist and exit the script(full page cache)
  2.  Tea::cache('front')->get();
  3.  
  4.  //Start recording and cache the page.
  5.  Tea::cache('front')->start();
  6.  $this->view()->render('index',$data);
  7.  Tea::cache('front')->end();
Partial cache can be used with the template engine:

  1.  $cacheOK = Tea::cache('front')->testPart('latestuser');
  2.  //if cache exist, skip retrieving from DB
  3.  if(!$cacheOK){
  4.      //get list from DB if cache not exist.
  5.  }
  6.  $this->view()->render('index',$data);
  7.  
  8.  //in the Template file
  9.  <!-- cache('latestuser', 60) -->
  10.  <li>The list of results to be loop</li>
  11.  <!-- endcache -->
Using getOnly or getExcept for full page cache:
  1.  //Use it in index.php
  2.  $check = Tea::cache('front')->getOnly('/blog', 3600, true);
  3.  
  4.  //Start recording and cache the page.
  5.  if($check)
  6.      Tea::cache('front')->start();
  7.  
  8.  Tea::app()->run();
  9.  
  10.  if($check)
  11.      Tea::cache('front')->end();
The example above caches all pages under /blog URL.

		getExcept (line 213)
	
 
	
	

Retrieve the full page caches which are not in the exception list (one/array of URLs).


You can retrieve a page cache by passing a url as defined in the routes.  Pass in an array of URLs if you need to output the cache NOT in those URLs  
  1. // All the URLs not found in the list will be retrieve
    2. 

  2.  Tea::cache('front')->getExcept('/blog', 3600);
    3. 

  3.  Tea::cache('front')->getExcept(array('/allurl','/gen_site', '/blog'), 3600);
    4. 

To output cache NOT in a certain URL(s) recursively, just pass in a True:  
  1.  // URLs in blog: /blog, /blog/article/:pname, /blog/archive/:year/:month
    2. 

  2.  // This will output all caches NOT start with /blog
    3. 

  3.  Tea::cache('front')->getExcept('/blog', 3600, true);
    4. 

If you need to output a list of cache NOT in the URLs recursively, just pass in an array:  
  1.  // This will output all cache NOT in URLs
    2. 

  2.  // Only /blog & /gen_site will be check recursively.
    3. 

  3.  Tea::cache('front')->getExcept(array('/allurl','/gen_site', '/blog'), 3600, array('/blog', '/gen_site'));
    4.
