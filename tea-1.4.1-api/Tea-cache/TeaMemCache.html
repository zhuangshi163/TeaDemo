TeaMemCache

Description
Description | Vars (details) | Methods (details)

TeaMemCache provides caching methods utilizing the Memcache extension.

If you have multiple servers for memcache, you would have to set it up in common.conf.php

  1.  // host, port, persistent, weight
  2.  $config['MEMCACHE'] = array(
  3.                        array('192.168.1.31', '11211', true, 40),
  4.                        array('192.168.1.23', '11211', true, 80)
  5.                      );
